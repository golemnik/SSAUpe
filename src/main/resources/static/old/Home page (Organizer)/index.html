О проекте:

Информационная система разработана в рамках курсового проектирования по дисциплине «Программная инженерия» на базе кафедры программных систем Самарского университета (Институт информатики и кибернетики). Проект реализует двухролевую модель доступа (Администратор / Волонтёр) и предназначен для автоматизации волонтёрской деятельности в соответствии с законодательством РФ.

Правовая информация:

Вся информация носит ознакомительный характер в рамках учебного проекта.
